04 - Qual o propósito do arquivo .gitkeep?
R:É uma prática padronizada adicionar um arquivo gitkeep dentro das pastas vazias para manter o controle dessas pastas

.gitignore - é usado para especificar quais arquivos devem ser ignorados pelo Git.
.git - armazena o histórico de commits do repositório. Este arquivo é necessário para que o Git funcione corretamente.
.gitmodules - Especifica submódulos Git que são usados pelo projeto.
.gitconfig - Armazena as configurações do Git para o usuário ou repositório.
.gitattributes - Define atributos específicos para arquivos ou diretórios.
